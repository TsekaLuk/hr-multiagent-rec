# 文献综述要点（论文第 1-2 章八股素材）

> 本文档提供每篇关键文献的「贡献 + 局限 + 与本工作关系」的 80–120 字总结，
> 用于第 1 章「国内外研究现状」和第 2 章「相关技术」的论文段落起草。

## 国外人岗匹配主线

### Zhu et al., 2018 — PJFNN [1]
**贡献**：首次提出 Person-Job Fit 联合表示学习神经网络，将简历与岗位的文本通过 CNN+多层感知机映射到共享空间，开启了「表示学习驱动的人岗匹配」范式。
**局限**：依赖人工标注的「成功匹配」样本，对同义表达和隐含能力识别能力有限。
**与本工作**：作为「单模型语义匹配」基线之一。

### Qin et al., 2018 — APJFNN [2]
**贡献**：提出能力感知（Ability-aware）神经网络，在岗位能力项与求职者经历之间建立细粒度对齐。
**局限**：能力项需要从岗位描述中规则抽取，对中文长尾岗位泛化能力弱。
**与本工作**：启发了我们的双向评分中「岗位侧 = 必需技能覆盖 × 硬门槛」结构。

### Bian et al., 2019, 2020 — Domain-PJF & MV-CoTeach [3,4]
**贡献**：研究跨招聘领域的迁移学习（Domain-PJF）与稀疏交互下的多视图协同教学（Multi-View Co-Teaching）。
**局限**：均面向英文/混合数据，未在中文招聘场景验证。
**与本工作**：稀疏标签问题在本课题中由「合成 + 真实交互」共同缓解。

### Lavi et al., 2021 — conSultantBERT [5]
**贡献**：将 Sentence-BERT 微调为 Siamese 结构匹配简历与岗位。
**局限**：使用通用预训练 BERT，未利用大语言模型的推理能力。
**与本工作**：本工作的语义编码层是 conSultantBERT 的「同族多语言升级版」。

### Yu et al., 2024, 2025 — ConFit & ConFit v2 [6,7]
**贡献**：将对比学习引入 PJF，通过数据增强、HyRe（假设简历嵌入）和 Runner-Up 难负样本挖掘提升排序效果，在公开 benchmark 上 SOTA。
**局限**：训练成本较高，且未涉及 Agent 协同与可解释性。
**与本工作**：是最直接的对比对象；ConFit-v2 的 HyRe 思想启发了我们的 Job-Analyst Agent 「先抽取再匹配」流程。

### Cao et al., 2024 — TAROT [8]
**贡献**：层次化多任务联合预训练，统一处理简历的半结构化字段。
**局限**：模型规模大，部署门槛高。
**与本工作**：本工作选择小规模 Qwen3-0.6B + 同族 Reranker，参数总量小 10×。

### Chen et al., 2024 — Job-SDF [9]
**贡献**：发布千万级中文岗位语料，附技能需求时序基准。
**局限**：实体名脱敏，仅保留 ID。
**与本工作**：作为辅助数据，提供技能本体与时间序列基线。

### Rosenberger et al., 2025 — CareerBERT [10]
**贡献**：将简历与 ESCO 职业体系映射到共享嵌入空间，MRR@100 = 0.328。
**局限**：单向（岗位 → 职业类别）映射，未做双向偏好建模。
**与本工作**：本工作借鉴「共享嵌入空间」思想并将其扩展到「同族模型族」。

## 多智能体推荐前沿

### Wang et al., 2024 — MACRec [14]
**贡献**：SIGIR 2024 工作，提出多智能体协同推荐框架，包含分析、检索、决策、反思等角色。
**局限**：通信开销大，未优化效率。
**与本工作**：本工作的 4 Agent 设计直接借鉴 MACRec 的「角色专职化」思想，但通过「结构化证据通信」削减了 70%+ 通信开销。

### Peng et al., 2025 — A Survey on LLM-powered Agents for Recommender Systems [15]
**贡献**：将 Agent 推荐系统综述为「推荐导向、交互导向、仿真导向」三类范式，定义画像-记忆-规划-行动四模块。
**局限**：综述性质，无具体方案。
**与本工作**：本工作属于「推荐导向 + 解释导向」结合范式。

### Chen et al., 2025 — Optima [16]
**贡献**：将「效率 + 效果」共同优化纳入多 Agent 评估。
**局限**：基准任务未涉及推荐。
**与本工作**：本工作通过「Top-M 截断 + 同族模型」实现 Optima 倡导的轻量化目标。

## 国内研究

### 梁婷婷 et al., 2020 — 本科层次大数据人才需求可视化分析 [11]
**贡献**：基于十几万条招聘信息分析本科层次大数据人才市场结构。
**与本工作**：提供「真实招聘数据具有明显结构规律」的现实依据。

### 朱瑜 et al., 2023 — BATPJF 端到端人岗匹配 [12]
**贡献**：基于深度学习的端到端中文 PJF 模型。
**局限**：单模型架构，无 Agent 协同。
**与本工作**：作为中文 PJF baseline 对比。

### 段雪艳 et al., 2025 — BERT + LightGBM 人岗匹配 [13]
**贡献**：BERT 语义 + 梯度提升融合，在真实竞赛数据集上验证中文语义表示有效性。
**局限**：未引入 LLM 与多 Agent。
**与本工作**：作为「单 BERT 基线」对照。
